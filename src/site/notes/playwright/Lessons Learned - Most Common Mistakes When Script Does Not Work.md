# Lessons Learned - Most Common Mistakes When Script Does Not Work

## Lessons Learned - Most Common Mistakes When Script Does Not Work

- missing `await` in statement
- `conts` or `let` - make sure you use the correct one 
- missing `async` or it should not be used here
- missing semicolon
- other typos
